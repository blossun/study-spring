인덱스 핸들러

* 다른 리소스에 대한 링크 제공
* 문서화

```java
@GetMapping("/api")
    public ResourceSupport root() {
        ResourceSupport index = new ResourceSupport();
        index.add(linkTo(EventController.class).withRel("events"));
        return index;
    }

```
