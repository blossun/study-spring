## WebClient 커스터마이징

`builder`와 `build()`를 호출하기 전에 커스터마이징 할 수 있다.

#### 로컬 커스터마이징

```java
@Override
public void run(ApplicationArguments args) throws Exception {
    WebClient webClient = builder
            .baseUrl("http://localhost:8080")
            .build();
}
```



#### 글로벌 커스터마이징

* `WebClientCustomizer`를 빈으로 등록해주면 된다.
